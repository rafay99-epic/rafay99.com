## Project Overview

**Dienstleisto** is a cross-platform job-seeking portal that connects job seekers with employers. The app features category-based job filtering, allowing users to browse and apply for positions across different industries and roles.

## Key Features

- **Job Listings**: Browse available positions with detailed descriptions and requirements
- **Category Filtering**: Filter jobs by industry, role type, location, and other criteria
- **Application Flow**: Apply to jobs directly through the app
- **Cross-Platform**: Built with Flutter for both Android and iOS

## Technical Architecture

- **Frontend**: Flutter mobile application handling the UI and user interactions
- **Backend**: Laravel API providing job data, authentication, and application management
- **Communication**: RESTful API connecting the Flutter client to the Laravel server

## Tech Stack

- **Mobile**: Flutter with Dart
- **Backend**: Laravel (PHP)
- **API**: RESTful endpoints

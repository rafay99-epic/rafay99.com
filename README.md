## Quick Start

```bash
git clone https://github.com/rafay99-epic/Astro-Portfolio-Blog.git
cd Astro-Portfolio-Blog
bun install
bun run dev
# open http://localhost:4321
```

## Project Layout

```
src/
  components/
    AstroComponent/       # Static components (zero JS)
    ReactComponent/       # Interactive React islands
  config/                 # Feature flags, theme tokens
  content/
    blog/                 # MDX blog posts
    projects/             # MDX project pages
  layouts/                # Page layouts
  pages/                  # Routes (blog, projects, about, contact, search...)
  styles/                 # Global styles
  types/                  # Shared TypeScript types
docs/                     # Project documentation
```
